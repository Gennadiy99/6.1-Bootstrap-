При подключении в "head" свойства "link" можно установить иконку сайта, которую будет видно в закладках. Для этого при наборе "link" нужно выбрать из списка "link:favicon" и добавится шаблонная строка: <link rel="shortcut icon" href="favicon.ico" type="image/x-icon"> в которой нужно указать путь к картинке которая находится в папке проэкта: href="img/favicon.ico"

При подключении стилей нужно свой файл style.css подключать в самом низу что бы он имел приоритет перед другими css файлами.

О создании разметки на "Bootstrap" можно посмотреть в Документации в разделе "Макеты - Layout" - "Grid"

Начинаем писать разметку:

  • <header> - тег в которм содержится верхняя часть сайта
  • <div class="container"> </div> - Обязательный класс Bootstrap который используется как ограничитель сайта по ширине.
  • <div class="row"> </div> - Отвечает в "Botstrap" за создание строки
  • <div class="col"> </div> - Отвечает за создание колонок

Сетка Bootstrapa расчитана на 12 столбцов, когда пишем col9 - Это означает что одна колонка занимает 9 столбцов, значит остается еще 3 свбободных для следующей колонки.

Пример кода:

            <header>
              <div class="container">
                <div class="row">
                  <div class="col12"> Column </div>
                </div>
              </div>
            </header>

В документации сайта в разделе " Контент" - "Типографика" описываются разные классы на "Bootstrap"

Цвета описываются в разделе "Утилиты" - "Цвета"

Об кнопках описывается в "Компоненты" - "Кнопки"

Очень интересное использование свойства для "border" которое описывается в CSS.

  • border-style: solid; - Устанавливает стиль начертания границы (сплошная)
  • border-width: 10px 0px 10px 30px; - Устанавливает ширину границы Верх Право Низ Лево (от центра)
  • border-color: red green blue yellow - Устанавливает цвет для каждой из границ
  • border-color: transparent ... - Устанавливает прозрачный цвет границы

Спицифика работы с маодальным окном "Bootstrap-5" Раздел "Компоненты" - "Модальные" Обязательно нужно связать модальное окно с кнопкой которая его вызывает. Для этого в кнопке нужно указать 2 атрибутa:

  1. data-bs-toggle="modal"
  2. data-bs-target="#exampleModal" - У кнопки и у Модального окна должен быть один id. Если менять то 2х местах нужно поменять название.

6.5 Аккордеон

В этом уроке я расскажу Как сделать Аккордеон на Bootstrap 5 любой сложности. Аккордеон позволяет нам скрывать и показывать различные элементы на сайте. Особенность Аккордеона в том, что одновременно может быть открыт только один элемент, а все остальные скрываются. Сделать Аккордеон на Bootstrap 5 очень просто, посмотрите полностью этот видео урок и вы узнаете как сделать аккордеон на Bootstrap 5, как настроить и стилизовать аккордеон и добавить новые элементы в аккордеон на Bootstrap 5.

Документация Bootstrap 5 по Аккордеону

По Аккордеону находится информация в "Компонеты" - "АККОРДИОН"

Материал урока:

Напоминание, что использовать цвет фона для нескольких секций, можно через добавления в стилях класса через запятую, оба класса будут иметь один стиль фона: 

            .about, .shedule {
                background-color: #eee;
              }

Создаем "Секцию" и обычную разметку Bootstrap, Для "Аккардиона" создадим колонку "col-md-4 col-sm-12" что бы на сред.экране было 3колонки а на мал.экр была Одна колонка. И в эту колонку будем вставлять код "Аккордиона" Пример разметки: 

              <section class="shedule">

                  <div class="row">
                    <div class="col-md-4 col-sm-12">

                        Модуль 1 <br/>
                        (HTML)
                      </h4>
                      ЗДЕСЬ БУДЕТ КОД АККОРДИОНА.
                    </div>
                  </div>
              </section>

Некоторый Синтаксис написани кода Bootstrap:

  • container - контейнер ограничивающий ширину
  • row - Строка, посути блок в котором расположены элем.
  • col-md-4 - Колонка, с указанием ее Обьема -4 при Размере экрана md
  • text-center - Стиль равен= "text-align:center;" выравнивание текста по середине.
  • text-uppercase - Текст в верхнем регистре.
  • mb-3 - Стиль равен= "margin-bottom:3px;" 3px - это примерая величина отступа.
  • mt-3 - Стил равен= "mrgin-top:3px;"
  • fw-bold - Стиль равен= font-weight: bold;
  • var(--bs-black) - bootstrap класс цвета "черный"
  • var(--bs-white) - bootstrap класс цвета "белый"
  • fst-italic - bootstrap класс устанавливает текст курсивом

При добавлении кода "Аккордиона" с сайта, сейчас выбираем 2-й вид.

Загрузив код видно уже работающий Аккордион. Но мы можем его корректировать.
Допустим, при нажатии на Аккордион меняется цвет его фона, это можно изменить переопределением стилей этого элемента.

Для этого необходимо открыть "Панель разработчика" на создаваемой странице, и навести на элемент который меняет цвет при наведении. После в закладке "Стили" скопировать силектор отвечающий за цвета фона .accordion-button:not(.collapsed){...} и поместить его в свой "css" файл для редактирования.

В файле "css" в скопированном классе, мы прописываем новые свойства цвета текста и цвета фона кнопки 

            .accordion-button:not(.collapsed){
              color: var(--bs-black);
              background-color: var(--bs-white);
            }

Тем самым мы переопределили Bootstrap классу ".accordion-button:not(.collapsed)" свои свойства Цвета и Фона.

Так же можно изменить и цвет "стрелочки" которая является svg иконкой. Для этого нужно выделить стрелочку, скоприровать код svg иконки. В коде svg иконки есть значение fill его меняем на white - белый. И после этого стрелочка должна стать белого цвета.

Данное свойство изменить не смог так как у меня в коде нет url иконки (стоит какая то переменная) Жду ответ а учителя.

Изменение стиля, делаем в Аккордионе, что бы один элемент был открытый изначально.
Делается это в html файле. Мы видим скопированный Аккордион в котором каждый элемент начанается с <div class="accordion-item"> </div>
в нутри аккордиона есть кнопка "botton" в которой указывается Название этой кнопки (меняем под необходимое).
Сам текс который появляется при нажатии, указывается в <div class="accordion-body" >

Есть важные связи между элементами Аккордиона

В первом div аккордиона есть атрибут id (Название аккордиона: id="accordionFlushExample") который связан с всеми вложенными в него элементами
<div class="accordion accordion-flush" id="accordionFlushExample"> </div>
в каждом элементе есть div с атрибутом в котором указан этот родительский id data-bs-parent="#accordionFlushExample"
Если менять название акордиона, то нужно менять его во всех атрибутах data-bs-parent каждого элемента.

Следующая связь через id между: тегом h2 в котором находится кнопка bottom с заголовком
<h2 class="accordion-header" id="flush-headingOne"> </h2>
И div с артрибутом aria-labelledby="flush-headingOne" в котором находится div с текстом видный при нажатии на кнопку.
По этому если менять связь между заголовком кнопки и текстом то делать это в id и атребуте одновременно.

Для того чтобы сделать Аккардион расскрытым по умолчанию нужно:

  • Убрать в "h2" в "bottom" класс "collapsed"
  • А у "div" c классами class="accordion-collapse collapse" (в который далее вложен текст) вставить класс "show"

Так же связаны между собой Кнопка и DIV с текстом. Через атрибуты в кнопке:
data-bs-target="#flush-collapseOne"
aria-controls="flush-collapseOne"
И через "id" в DIV:
id="flush-collapseOne"
Если менять название атрибутов то вместе с id
