Подключить "Bootstrap" можем зайдя на сайт Bootstrap через "CDN" скопировав ссылку "link" и всатвив ее на страницу html в hade: Перед своим стилевым файлом.

Так же подключаем "java-script" скопирвавав ссылку script И вставив ее в канце станицы перед закрытием тега "body"

На сайте интересует страница скачать "Download": Что бы все возможности Bootstrap работали полностью, в разделе:
"Исходные файлы" - "Source files"
нажимаем кнопку "скачать" - "download source",

После скачивания архиированной папки "bootstrap-5.3.3" паспаковываем ее и копируем из папки "dist" нужные файлы:

  • В папке "js" - файл "bootstrap.bundle.min"
  • В папке "css" - файл "bootstrap.min"

Скопированные файлы ложим в проэкт в соответствующие папки "js" / "css".

Теперь мы подключили локально "bootstrap js / css" и можно удалить в "head" путь к bootstrap на сайте и указать путь к созданному файлу в проэкте.

для этого в "link" удаляем значение ссылки, и вставляем туда путь к своему файлу.

  • Удаляем: href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css
  • Вставляем путь: href="css/bootstrap.min.css"

Все остальные атрибуты в "link" удаляются, кроме Атрибута:

  • rel="stylesheet"

Также подключим локальный файл "js" удалим в script src: путь к сайту и вставим путь к локальному файлу:

  • Удалим: src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
  • Вставим: src="js/bootstrap.bundle.min.js"

Все остальные атрибуты убираем.
