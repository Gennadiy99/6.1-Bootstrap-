6.5 Аккордеон

В этом уроке я расскажу Как сделать Аккордеон на Bootstrap 5 любой сложности. Аккордеон позволяет нам скрывать и показывать различные элементы на сайте. Особенность Аккордеона в том, что одновременно может быть открыт только один элемент, а все остальные скрываются. Сделать Аккордеон на Bootstrap 5 очень просто, посмотрите полностью этот видео урок и вы узнаете как сделать аккордеон на Bootstrap 5, как настроить и стилизовать аккордеон и добавить новые элементы в аккордеон на Bootstrap 5.

Документация Bootstrap 5 по Аккордеону

По Аккордеону находится информация в "Компонеты" - "АККОРДИОН"

Материал урока:

Напоминание, что использовать цвет фона для нескольких секций, можно через добавления в стилях класса через запятую, оба класса будут иметь один стиль фона: 

            .about, .shedule {

Создаем "Секцию" и обычную разметку Bootstrap, Для "Аккардиона" создадим колонку "col-md-4 col-sm-12" что бы на сред.экране было 3колонки а на мал.экр была Одна колонка. И в эту колонку будем вставлять код "Аккордиона" Пример разметки:

                      </h4>
                      ЗДЕСЬ БУДЕТ КОД АККОРДИОНА.
                    </div>
                  </div>

Некоторый Синтаксис написани кода Bootstrap:

  • container - контейнер ограничивающий ширину
  • row - Строка, посути блок в котором расположены элем.
  • col-md-4 - Колонка, с указанием ее Обьема -4 при Размере экрана md
  • text-center - Стиль равен= "text-align:center;" выравнивание текста по середине.
  • text-uppercase - Текст в верхнем регистре.
  • mb-3 - Стиль равен= "margin-bottom:3px;" 3px - это примерая величина отступа.
  • mt-3 - Стил равен= "mrgin-top:3px;"
  • fw-bold - Стиль равен= font-weight: bold;
  • var(--bs-black) - bootstrap класс цвета "черный"
  • var(--bs-white) - bootstrap класс цвета "белый"
  • fst-italic - bootstrap класс устанавливает текст курсивом
  • mt-sm-5 - (margin-top) на (sm) - малом экране = 5
  • mb-sm-5 - (margin-bottom) на (sm) - малом экране = 5

При добавлении кода "Аккордиона" с сайта, сейчас выбираем 2-й вид.

Загрузив код видно уже работающий Аккордион. Но мы можем его корректировать.
Допустим, при нажатии на Аккордион меняется цвет его фона, это можно изменить переопределением стилей этого элемента.

Для этого необходимо открыть "Панель разработчика" на создаваемой странице, и навести на элемент который меняет цвет при наведении. После в закладке "Стили" скопировать силектор отвечающий за цвета фона .accordion-button:not(.collapsed){...} и поместить его в свой "css" файл для редактирования.

В файле "css" в скопированном классе, мы прописываем новые свойства цвета текста и цвета фона кнопки 

            .accordion-button:not(.collapsed){
              color: var(--bs-black);
              background-color: var(--bs-white);
            }

Тем самым мы переопределили Bootstrap классу ".accordion-button:not(.collapsed)" свои свойства Цвета и Фона.

Так же можно изменить и цвет "стрелочки" которая является svg иконкой. Для этого нужно выделить стрелочку, скоприровать код svg иконки. В коде svg иконки есть значение fill его меняем на white - белый. И после этого стрелочка должна стать белого цвета.

Данное свойство изменить не смог так как у меня в коде нет url иконки (стоит какая то переменная) Жду ответ а учителя.

Из-за разных версий "Bootstrap" разный подход в описании данного стиля. В случаи если в стиле указывается не "url" иконки а переменная color: var(--colorWhite); background-color: var(--bgc1); 

            .accordion-button:not(.collapsed) {
              color: var(--colorWhite);
              background-color: var(--bgc1);
            }

Нужно зайти в файл бутстрапа "css" найти там переменнную var(--colorWhite) Скопировать из нее стиль иконки и перенести в свой файл "css". Там указать класс, скопированный из панели разработки при наведении, и в свойстве becground: (указать) url: (вставить скопированный стиль иконки из переменной будстрапа). В самой переменной в свойстве "fill" поменять значение на "нужный цвет". И переопределение цвета иконки готово.

Изменение стиля, делаем в Аккордионе, что бы один элемент был открытый изначально.
Делается это в html файле. Мы видим скопированный Аккордион в котором каждый элемент начанается с <div class="accordion-item"> </div>
в нутри аккордиона есть кнопка "botton" в которой указывается Название этой кнопки (меняем под необходимое).
Сам текс который появляется при нажатии, указывается в <div class="accordion-body" >

Есть важные связи между элементами Аккордиона

В первом div аккордиона есть атрибут id (Название аккордиона: id="accordionFlushExample") который связан с всеми вложенными в него элементами
<div class="accordion accordion-flush" id="accordionFlushExample"> </div>
в каждом элементе есть div с атрибутом в котором указан этот родительский id data-bs-parent="#accordionFlushExample"
Если менять название акордиона, то нужно менять его во всех атрибутах data-bs-parent каждого элемента.

Следующая связь через id между: тегом h2 в котором находится кнопка bottom с заголовком
<h2 class="accordion-header" id="flush-headingOne"> </h2>
И div с артрибутом aria-labelledby="flush-headingOne" в котором находится div с текстом видный при нажатии на кнопку.
По этому если менять связь между заголовком кнопки и текстом то делать это в id и атребуте одновременно.

Для того чтобы сделать Аккардион расскрытым по умолчанию нужно:

  • Убрать в "h2" в "bottom" класс "collapsed"
  • А у "div" c классами class="accordion-collapse collapse" (в который далее вложен текст) вставить класс "show"

Так же связаны между собой Кнопка и DIV с текстом. Через атрибуты в кнопке:
data-bs-target="#flush-collapseOne"
aria-controls="flush-collapseOne"
И через "id" в DIV:
id="flush-collapseOne"
Если менять название атрибутов то вместе с id

6.12 Адаптивный сайт, Создание своей сетки

В этом уроке мы будем учиться создавать адаптивную сетку на CSS. Так же научимся работать с переменными и функцией calc. Узнаем как создавать медиа запросы, чтобы сделать адаптивную версию своего сайта для мобильных устройств. Научимся создавать макет с 12-колончатой сеткой в Photoshop, сравним нашу сетку с сеткой Bootstrap 5, а так же научимся её подключать и узнаем как проверять свой сайт на эмуляторе мобильных устройств прямо с браузера на ПК.

Скачать готовые файлы

Для создания адаптивного сайта, можно использовать не весь "Bootstrap" а только частичные его элементы, что бы не скачивать весь функционал "Bootstrap", который мы не будем искпользовать.

некоторые свойства "css" для стилизации "html"

  • (для всех Тегов и псевдоэлементов) наследование свойства "box-sizing" от родительского элемента (html), из-за свойства - "inherit" 
                    *, *:before, *:after{
                  box-sizing: inherit;
                }
  • Свойство box-sizing:border-box; (html): 
                    html {
                  box-sizing: border-box;
                  font-size: 18px;
                }
  • Сброс отступов. 
                    body,p,h1,h2,h3,ul,ol{
                  margin: 0;
                  padding: 0;
                }
  • Класс Флекc элементы, стоят в рад. 
                    .row{
                  display: flex;
                  flex-wrap: wrap;
                }
    разрешон перенос.
  • Для всех "div" внутри "row" стили: 
                    .row div{
                  background-color: wheat;
                  padding: 15px;
                  border: 1px solid gray;
                  margin: 0 auto;
                }
    Свойство "margin" выравнивает элем по центру.
  • Функция calc вычисляет ширину от Родительского элемента. 
                    .col-3 {
                  width: calc(25% - 15px);
                }
                а -15px из ширины эл. уменьшая эл. дают возможность уст. margin так как он "auto" и занимает все свободное место.
  • Можно создать переменную для гибкой настройки отступа между элем. 
                    :root{
                  --gutter: 20px;
               }
              
                    .col-3 {
                  width: calc(25% - var(--gutter));
                }
                .col-6 {
                  width: calc(50% - var(--gutter));
                }
                .col-12 {
                  width: calc(100% - var(--gutter));
                }

Как Правильно писать медиазапросы для адаптивной верстки

Медиазапросы пишутся в конце всех стилей или в отдельном файле и подключаются после своего файла "CSS"

Помним о принципе в "CSS" те свойства, что указаны для элемента изначально, применяется и далее если их не переопределить.

В файле "CSS" нужно указывать стили экрана от большего к меньшему.

Что бы управлять адаптивностью элементов через классы, нужно создать классы, в медиазапросе и они будут работать только при этой ширине экрана. 

            @media screen and (max-width: 768px) {
              .col-md-3 {
                width: calc(25% - var(--gutter));
              }
              .col-md-4 {
                width: calc(33.33% - var(--gutter));
              }
              .col-md-6 {
                width: calc(50% - var(--gutter));
              }
              .col-md-12 {
                width: calc(100% - var(--gutter));
              }
            } 
            Теперь эти классы можно применят к элементам и они будут срабатывать только при указанной в "Медиа-запросе",- ширине экрана.

Так же нужно помнить, что можно использовать отделные элементы из "Bootstrap" и скачивать то что вы хотите использовать, к примеру можно скачать только сетку Bootstrap и использовать только ее.

  • Для этого нужно загрузить с сайта Bootstrap весь каталог
  • Зайти папку "dist"
  • Потом в папку "css"
  • И найти файл bootstrap-grid-min.css
  • Скопировать его и перенести его к себе в папку "css"

В Figma можно воспользоваться линейкой нажав - "shift+r"
Зажав левой клавишей на линейке можно перетянуть линию к элементу и увидеть число в пикселях. Таким образом можно замерять расстояние до элемента, между элементами.

Интерессная особенность в СБРОСЕ у последнего элемента "margin-bottom" для того что бы находящиеся в блоке элементы не сдвигались в нутри блока ввер, а зазор между ними оставался, нужно последнему элементу убирать свойство:"margin-bottom"
Вот пример кода описанный в стилях: 

            .burger__line {
              width: 21px;
              height: 3px;
              display: block;
              background-color: #fff;
              margin-bottom: 4px;
            }
            .burger__line:last-child{
              margin-bottom: 0;
            }

В "Figma" при инспектировании стиля можно увидеть свойство: "letter-spacing: 2%"
Что бы правильно перенести это свойство, нужно открыть "расширение" - "Inspect Styles" для этого элемента, и там скоприровать свойство: letter-spacing: 0.02em;. Оно будет уже указанно в правильных еденицах измерения.

Бывает ситуация когда по макету "Figma" нижний отступ между элементами "margin:35px" а в сравнении с скрином экрана видим разницу. В этом случаи нужно зацти в "Панель разработчика" в раздел где отрисованы отступы и в нем менять значение нижнего "margin" пока он совпадет с скрином, после чего внести это значение в код: "margin: 17px;"
